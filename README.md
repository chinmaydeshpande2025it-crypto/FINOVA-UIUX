# FINOVA-UIUX
 Finova is a user-friendly financial management tool designed to help users track income, expenses, and budgets. It offers features like dashboards, reminders, and reports. Built using Figma, it focuses on simple UI/UX design to make financial management easy and efficient.

# Problem Statement

Many individuals face difficulties in managing personal finances due to:

Lack of proper expense tracking
Poor budgeting habits
Complex and confusing financial apps
No reminders for bills or payments
Difficulty understanding financial data

Existing solutions are often too complicated or lack clarity, making them unsuitable for everyday users.

# Solutions Provided by the App

The Finova app solves these problems by:

Providing easy expense and income tracking
Offering categorized transactions
Displaying clear dashboard summaries
Using charts and graphs for insights
Allowing budget setting and monitoring
Adding bill reminders and notifications
Supporting savings goals and EMI tracking
Ensuring simple navigation and clean UI

This makes financial management easy, organized, and accessible.
